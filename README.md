# Lab Tracker

Канбан-доска для отслеживания лабораторных работ. Позволяет держать все задания в одном месте, видеть дедлайны и понимать, что горит, а что терпит.

---

## Ключевые особенности

- **Drag & Drop** — перетаскивание карточек между колонками мышью и тачем.
- **Индикатор срочности** — цветная полоса на каждой карточке: зелёная (больше 7 дней), жёлтая (3–7 дней), красная (меньше 3 дней). При менее чем 24 часах до дедлайна полоса пульсирует.
- **Подзадачи** — разбивка работы на шаги с прогресс-баром прямо на карточке.
- **Боковая панель** — открывается кликом по карточке. Три вкладки: подзадачи с чекбоксами, чат-лог (заметки + системные записи о перемещениях), текстовое описание.
- **Пользовательские колонки** — можно добавить до 8 колонок; системные (Не начато / В работе / Сдано) защищены от удаления.
- **Фильтр и сортировка** — фильтр по названию предмета, сортировка по дедлайну или дате создания.
- **Режим фокуса** — скрывает все колонки кроме одной, чтобы не отвлекаться.
- **Тёмная и светлая тема** — переключается кнопкой в шапке, выбор сохраняется.
- **Клавиатурные шорткаты** — `Ctrl+N` создать задачу, `Ctrl+F` фокус на поиске, `Escape` закрыть панель.
- **Локальное хранилище** — данные сохраняются в `localStorage`; при недоступности хранилища показывается предупреждение.

## Технический стек

| Слой | Технология |
|---|---|
| UI | React 18 |
| Сборка | Vite 6 |
| Стили | Tailwind CSS 3 |
| Drag & Drop | @dnd-kit/core + @dnd-kit/sortable |
| Персистентность | localStorage (нет бэкенда) |

## Структура проекта

```
├── index.html                        — HTML-точка входа
├── vite.config.js                    — конфиг сборщика
├── tailwind.config.js                — настройки Tailwind (тема, accent-цвет)
├── postcss.config.js                 — PostCSS
└── src/
    ├── main.jsx                      — монтирование React-приложения
    ├── index.css                     — глобальные стили и Tailwind-директивы
    ├── App.jsx                       — корневой компонент: тема + провайдер контекста
    │
    ├── context/
    │   └── TaskContext.jsx           — глобальное состояние (задачи, колонки, тосты);
    │                                   CRUD-операции: createTask, updateTask, deleteTask,
    │                                   moveTask, createColumn, deleteColumn
    │
    ├── components/
    │   ├── Board/
    │   │   ├── Board.jsx             — главный layout: шапка, тулбар, DnD-контекст,
    │   │   │                           боковая панель, фильтр, сортировка, фокус-режим
    │   │   ├── Column.jsx            — droppable-колонка с заголовком и списком карточек;
    │   │   │                           диалог удаления с выбором судьбы задач
    │   │   └── AddColumnPanel.jsx    — кнопка/форма добавления новой колонки
    │   │
    │   ├── Card/
    │   │   ├── TaskCard.jsx          — sortable-карточка: urgency-полоса, предмет,
    │   │   │                           заголовок, прогресс подзадач, дедлайн, кнопка удаления
    │   │   └── UrgencyBadge.jsx      — отдельный бейдж срочности (используется в сайдбаре)
    │   │
    │   ├── Modal/
    │   │   ├── TaskModal.jsx         — форма создания и редактирования задачи
    │   │   │                           (название, предмет, дедлайн, статус, подзадачи)
    │   │   └── ConfirmModal.jsx      — универсальное модальное окно подтверждения
    │   │
    │   ├── Sidebar/
    │   │   └── TaskSidebar.jsx       — боковая панель задачи с тремя вкладками:
    │   │                               Подзадачи / Чат (activity log) / Описание (notes)
    │   │
    │   └── Toast/
    │       └── Toast.jsx             — стек всплывающих уведомлений
    │
    ├── hooks/
    │   ├── useLocalStorage.js        — хук для синхронизации состояния с localStorage
    │   └── useTheme.js               — хук переключения тёмной/светлой темы
    │
    └── utils/
        ├── mockData.js               — демо-задачи и колонки для первого запуска
        ├── urgency.js                — логика уровней срочности и анимации пульса
        └── time.js                   — форматирование относительного времени («5 мин назад»)
```
